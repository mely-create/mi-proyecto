# Biblioteca Digital

## Descripción

Este proyecto es una aplicación que permite gestionar libros de manera digital.

* Motivación: Aprender programación
* Problema: Organización de libros
* Solución: Sistema digital
* Aprendizaje: Uso de bases de datos

## Instalación

1. Clonar el repositorio:
   git clone https://github.com/tuusuario/mi-proyecto

2. Instalar dependencias:
   pip install pymongo

## Uso

Ejecutar el programa:
python main.py

## Tecnologías

* Python
* MongoDB
